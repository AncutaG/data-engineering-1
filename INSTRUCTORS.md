## Building the course content (slides and docs atm)

### top-level

From the top-level directory of project, you can:

- create all slides

    make

- clean up

    make clean


### a single week

From a week's folder (e.g., `02-Working-with-Data`), you can:

- create slides

    make

- clean up

    make clean

- view slides locally

    open <lecture-component>.html
