  • pull the image:

    docker pull midsw205/base

  • create your mids-w205 workspace:

    mkdir w205

  • (Mac)

    docker run \
  -it \
  --rm \
  -v /Users/<user>/w205:/w205 \
  midsw205/base:latest \
  bash

  • exit (or ctrl-d)
